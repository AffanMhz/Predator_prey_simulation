üî¨ The Science Behind It

Lotka-Volterra Equations

The mathematical foundation of this simulator comes from the Lotka-Volterra equations, developed independently by Alfred Lotka (1925) and Vito Volterra (1926). These equations describe the population dynamics of two interacting species:

Prey equation: dR/dt = Œ±R - Œ≤RF

Predator equation: dF/dt = Œ¥RF - Œ≥F

  • Œ± - Prey growth rate (reproduction without predators)
  • Œ≤ - Predation rate (interaction coefficient)
  • Œ≥ - Predator mortality rate (death without prey)
  • Œ¥ - Predator reproduction efficiency from eating prey

Grid-Based vs. Continuous Models

This simulator implements a discrete grid-based model where individuals move, eat, reproduce, and die according to probabilistic rules. This approach captures spatial effects and stochastic behavior that continuous differential equations cannot represent.

The comparison between the grid simulation and the Lotka-Volterra equations reveals when continuous approximations work well and when spatial and stochastic effects dominate.

HTML5 Canvas

Grid rendering and graph visualization using native canvas API for performance.

Vanilla JavaScript

Pure JavaScript implementation with no external dependencies - runs entirely in the browser.

Numerical Integration

4th-order Runge-Kutta method for solving differential equations with high accuracy.

Statistical Methods

Linear regression and grid search optimization for parameter estimation from simulation data.

üìù How to Use

  1. Open the simulator page
  2. Adjust parameters using the sliders in the control panel
  3. Click "Start / Pause" to run or pause the simulation
  4. Observe the spatial grid, time series, and phase space visualizations
  5. Click "Reset" to reinitialize with new random conditions
  6. Use "Compute equilibrium & Jacobian" for theoretical analysis
  7. Try "Estimate LV Parameters" to fit equations to simulation data
  8. Click "Export CSV" to save population history for external analysis
